# typical
 This can be used to use typical functionality of Tizen RT on ARTIK053.

## Enabled Feature
### File system
  MTD, ProcFS, SmartFS

### Kernel
  ADC, GPIO, I2C, PWM, RTC, SPI, UART, Watchdog  
  Debug  
  POSIX APIs  
  Workqueue

### Network
  ARP, Dhcp, FTP, ICMP, IGMP, Ipv4, Lwip, Ntpclient, TCP, Telnet, TLS, UDP, SMTP, WIFI, Wpa_supplicant
